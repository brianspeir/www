Emoji

Pictorial symbols for a word or phrase, reimagined for a simpler, more useful and more enjoyable world.

An exploration in “as little design as possible” this project simplifies and refines emoji in an effort to align them with the current aesthetic of mobile applications. Gone are the drop shadows. Smoothed are the gradients. Simplified are the micro details. Some names have been changed to better reflect the mood associated with the visual. Currently available as vector graphics. The goal of the project is to create a standardized, open source emoji font not beholden to a specific device or operating system.
